MessagingSystem

interface MessagingSystem : AssertsConsuming, PluggedSystem

Messaging abstraction for the testing system. Every messaging system more or less has the same behaviour. MessagingSystem defines an abstraction point for these tools. The popular implementation is Kafka, and Stove Testing supports it. As an example:

To set it up:

TestSystem().withKafka()
Content copied to clipboard

Usage in testing:

TestSystem.instance
.kafka()
.publish("test.topic", TestEvent(id = "id-test"))
.shouldBeConsumed(message = TestEvent(id = "id-test"))
.shouldBeConsumedOnCondition<TestEvent> { actual ->
    actual.id == "id-test"
}
